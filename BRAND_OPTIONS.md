# NCLEX-PN Brand / Name Options

Current positioning: NCLEX-PN daily trainer + error journal + remediation coach for Canadian RPN and US LPN students. Not an official NCLEX/NCSBN product and not a UWorld clone.

## Strongest options

1. **1Stop NCLEX PN**
   - Closest to the user suggestion.
   - Clear, direct, easy to remember.
   - Risk: "NCLEX" in the name may need legal/brand review and a very visible non-affiliation disclaimer.

2. **PN Pass Coach**
   - Simple and benefit-driven.
   - Good fit for daily plan + error journal.
   - Risk: avoid "guaranteed pass" claims; name is okay if marketing is careful.

3. **NurseReady PN**
   - Professional, expandable to RN later.
   - Good for Canadian/US audience without sounding like a cheap qbank.
   - Suggested structure later: NurseReady PN, NurseReady RN.

4. **PN Daily Trainer**
   - Very clear and honest.
   - Strong SEO/product clarity.
   - Less flashy, but trustworthy.

5. **PassPath PN**
   - Marketable, short, and expandable.
   - Good fit for remediation/daily plan.
   - Needs trademark/domain check.

6. **RPN/LPN Coach**
   - Immediately says who it is for.
   - Less cool, but highly clear for Canada + US.

## My current recommendation

Use a working product name for now:

**NurseReady PN**

Use the tagline:

**NCLEX-PN Daily Trainer + Error Journal**

Reason: "NurseReady PN" is more brandable and expandable than "1Stop NCLEX," while the tagline still captures the NCLEX-PN search intent. Later, the same system can become NurseReady RN without rebranding the whole platform.

## Legal/positioning note

If we put NCLEX directly in the app name, we need visible disclaimers:

- Not affiliated with NCSBN.
- Does not use real exam questions.
- NCLEX is a trademark of its owner.
- Provides NCLEX-style practice only.
